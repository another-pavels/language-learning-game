# Numbers Repetition Game

A simple language learning game as a demonstration of a basic React/Redux app.

[Demo](https://numbersgame.herokuapp.com)

## Getting Started

```sh
git clone https://github.com/maleksan/language-learning-game.git
cd language-game
npm install
npm run build
npm start
```
